What is inheritance?
Inheritance is the mechanism by which a class is created as an extension of another class retaining all its properties. The original class is called base class and new class is called derived class. 

e.g. If you have a class called human, you can create another class called student based on human class. 

Inheritance is one of the basic concepts of object oriented languages. 
 
What is the keyword "final"?
final keyword can be used in different contexts.

A final variable is the one which can not be modified. It is a constant. Its value is set either using an initializer or using an assignment statement. Once assigned, its value can not be changed. 

A final object can not change its reference. Its state can change - its field can change. But it can not refer to another object. 

A final method can not be overridden. The derived classes can not re-define a final method.

A final class can not be inherited - it can not be subclassed.

 
What is static keyword?
A static variable of a class is common to all objects of the class unlike an instance variable.

A static method of a class can be invoked without using object of a class. And a static method can access only static variables of a class. 
 
Can a class be defined as static?
A top level class can not be static. But an inner class - a nested class can be static in Java.
 
What is static block in Java?
Java static block or static initializer block is the group of statements that gets executed when the class is loaded  Java ClassLoader. 

Static block is used to initialize static variables of the class. 

A class can have multiple static blocks which are executed in the same order they are defined in the class. 

e.g.
   class A{
       static int m;
        int n;
 static{
          m = 1;
       }
       A(int num){
           n = num;
      }
   }
Here the block static { m = 1;} is the static initializer block. Unlike the constructor, this block gets executed only once. It initiailizes the static variable m
